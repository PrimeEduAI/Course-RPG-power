# AI 影片冒險者 — 設計方向

六小時兒童課程「用 AI 做影片」的課堂互動 3D 網站。核心隱喻：**上課 = 冒險，知識點 = 裝備，導演思維 = 天賦技能樹**。

## 概念

- 小朋友（全班共同）扮演一位 Q 版 RO 風格冒險者，一開始只穿初心者布衣。
- 老師每教完一個知識點，就在左上角「冒險背包」把該知識點打勾 → 對應裝備以 3D 動畫（從天而降、旋轉、粒子爆光、音效）穿到冒險者身上。
- 思維面的內容（導演對色彩、分鏡、節奏的思維）不是裝備，而是「導演天賦」技能樹：點亮節點後，冒險者身上會出現對應色系的光環粒子。
- 全部點完 → 冒險者從「新手冒險者」升級成「AI 大導演」（皇冠是最後一件裝備）。

## 呈現

- **舞台**：草地圓台 + 藍天 + 飄雲 + 小樹小蘑菇，OrbitControls 可拖曳環繞。
- **角色**：程式化低多邊形 chibi（大頭小身，MeshToonMaterial cel-shading），無外部模型檔 → 零授權問題、載入極快。
- **UI**：
  - 左上 🎒 冒險背包：章節分組的知識點清單（勾選 = 穿裝備）。
  - 右上 ✨ 導演天賦：SVG 技能樹覆蓋層，三分支（色彩之眼 / 分鏡之眼 / 節奏之心）+ 根節點導演之魂。
  - 頂部中央：稱號 + EXP 進度條（EXP = 已勾裝備 + 已點天賦）。
- **持久化**：localStorage，課程中途重新整理不會掉進度；背包面板內有重置鈕（開新班用）。
- **音效**：WebAudio 合成的叮聲/琶音，無音檔。

## 內容編輯器(/edit.html)

- 冒險頁右下角 ✏️ 進入;老師可直接編輯**章節/知識點(裝備)**與**導演天賦(技能樹)**,不用改程式。
- 可新增/刪除章節與知識點;每個知識點可挑 10 種 3D 模型(頭帶/皇冠/劍/盾/披風/胸甲/場記板精靈/靴/腰鈴/發光徽章)。
- 天賦固定「根節點 + 三分支」結構,分支名稱可改,每支 1~4 個節點,樹的座標自動排版。
- 按「儲存變更」寫入 localStorage(`ai-video-quest-content-v1`),冒險頁收到 storage 事件自動重新載入;沒有自訂內容時用 `src/content.ts` 的預設。
- 小朋友的打勾進度(另一個 key)在改內容後保留;被刪除的知識點 id 會在載入時自動清掉。

## 技術

- Vite + TypeScript + Three.js（唯一 runtime 依賴），bun 安裝與啟動。
- 課程文字全部集中在 `src/content.ts`（章節、裝備知識點、天賦節點），對照簡報改字即可，不用動 3D 程式。
- 裝備 id ↔ 3D 建模函式對應在 `src/equipment.ts`；新增知識點若無專屬模型可先掛通用徽章。

## 取捨備註

- 技能樹**不強制先點父節點**（老師上課順序可能跳動），連線只做視覺表現。
- 角色/裝備走程式化建模而非 GLB 素材：風格統一可控，缺點是精緻度上限較低；未來若要更精緻可換成 VRM/GLB 模型並保留同一套 attach 點。
